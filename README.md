# 🏦 Bank Account Manager (C# Console Application)

A simple console-based Bank Account Manager built with C#.

This project simulates a basic banking system where users can create accounts, deposit and withdraw money, search accounts, and manage account information.

---

## Features

- 👤 Create new bank accounts
- 📋 Show all accounts
- 💰 Deposit money
- 💸 Withdraw money
- 🔍 Search accounts by owner name
- 🗑 Delete accounts
- ✏️ Edit account owner name
- ⭐ Show rich accounts (balance over $2000)
